# 问题反馈指南

## 反馈前准备

在进行任何问题反馈前，请先阅读 [《提问的智慧》](https://github.com/ruby-china/How-To-Ask-Questions-The-Smart-Way/blob/master/README-zh_CN.md)

## 反馈渠道

### GitHub Issues（推荐）

- 地址：[https://github.com/WangNingkai/OLAINDEX/issues](https://github.com/WangNingkai/OLAINDEX/issues)
- 适用：Bug 反馈、功能建议、问题讨论

### GitHub Discussions

- 地址：[https://github.com/WangNingkai/OLAINDEX/discussions](https://github.com/WangNingkai/OLAINDEX/discussions)
- 适用：使用交流、经验分享

## 问题反馈模板

### Bug 反馈

请提供以下信息：

```markdown
**问题描述**
简要描述遇到的问题

**复现步骤**
1. 执行的操作
2. 出现的错误

**环境信息**
- PHP 版本：
- Laravel 版本：
- 服务器系统：
- Web 服务器：

**错误日志**
```
粘贴相关错误日志
```

**截图**
如有必要，请附上截图
```

### 功能建议

```markdown
**建议描述**
简要描述你希望添加的功能

**使用场景**
描述这个功能的使用场景

**期望效果**
描述你期望的实现效果
```

## 注意事项

- 请确保提供信息完整准确，否则可能无法跟进
- 反馈前请先搜索是否已有相同问题
- 保持言行举止文明，尊重他人
